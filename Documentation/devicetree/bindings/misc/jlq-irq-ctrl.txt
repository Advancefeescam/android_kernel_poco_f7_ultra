jr510 lpm modules edge irq to level irq controll device

Required properties:
- compatible = "jlq,lpm-irq-ctrl";
- reg: specify the lpm-irq-ctrl module base address  and associated address range
- irq-mappings: specify the mapped edge->level gic spi irq number, the sequence
  of the irq represent the bit offset from the base address of lpm-irq-ctrl
  module, if no irq is mapped, set relevent byte to 0


Example:
	lpm-irq-ctl@0x34510704 {
		compatible = "jlq,lpm-irq-ctrl";
		reg = <0x34510704 8>;
		irq-mappings = [
			75 76 77 78 79 7a 7b 7c
			7d 7e 7f 81 82 83 84 85
			86 87 88 89 8a 8b 8c 8d
			8e 95 96 97 99 9a 9b 9c
			b1 b2 b3 b4 b5 b6 b7 b8
			b9 ba bb bc bd be
		];
	};
